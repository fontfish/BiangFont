BiangFont in Use

A Simple Test

BiangFont is a micro-glyphset font made to allow on-screen display of the Chinese character 𰻞 (biáng, simplified: 𰻝), containing only this and a few other related characters. Many CJK typefaces do not display this character owing to its uncommon useage and very recent adoption into Unicode.

This font is also my first attempt at implementing variable font features. A demonstration of this is below. Please feel free to try changing the font weight using the sliders.

For more information about BiangFont, please see the GitHub page at https://github.com/fontfish/BiangFont
